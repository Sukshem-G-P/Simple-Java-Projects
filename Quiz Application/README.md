
Overview :<br>
QuizApplication is a simple Java-based console quiz program designed to test your general knowledge. It loads a set of multiple-choice questions from a text file, presents them to the user, and calculates the score based on the user's responses. The application supports various types of general knowledge questions, and the questions are presented in a straightforward format, making it easy to use.<br><br>
Features:<br>
Easy Question Setup: The questions are stored in a questions.txt file with a specific format, making it easy to modify or add new questions.<br>
Interactive Quiz: The application interactively asks questions, takes user input, and provides immediate feedback on whether the answer is correct or incorrect.<br>
Score Calculation: The application calculates the user's score based on the number of correct answers and provides a summary at the end.<br>
Simple and Intuitive Interface: The command-line interface is simple, making it suitable for users with basic Java knowledge.<br>
